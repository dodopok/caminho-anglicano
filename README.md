# Caminho Anglicano

Site do CaminhoAnglicano.com.br

A Nuxt 3 application built with Vue 3 and TypeScript.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start development server on `http://localhost:3000`:

```bash
npm run dev
```

## Production

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Type Checking

Run TypeScript type checking:

```bash
npm run typecheck
```

## Linting

Run ESLint:

```bash
npm run lint
```
